
Tradutor primitivo para a linguagem LolCode, desenvolvido como atividade de aprendizado durante a disciplina de Linguagens de Programação.
A tradução é realizada para um arquivo de código fonte da linguagem Python.
O conjunto completo de instruções da linguagem pode ser encontrado no link [lolCode.org](http://lolcode.org/).

Implementado utilizando ANTLR4.
